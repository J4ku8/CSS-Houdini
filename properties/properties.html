Properties and Values API

Custom Properties, extend CSS Variables to allow us to register properties and define their type, initial value, and determine inheritance.


Worklet example:

CSS.registerProperty({
name:'--color-hover',
syntax: <color>,
initialValue: 'transparent',
inherits: false });

What we have defined:

  1. name: is name of property which we then use as variable in css file
  2. syntax: defines how to parse property, could accept <integer>, <number>, <time>, etc..
  3. initialValue: this property is used if no value is passed in
  4. inherits: tells browser if should be inherit down to the DOM tree

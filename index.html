What is CSS Houdini

CSS Houdini stands for bringing CSS and JS closer together.
CSS Houdini is set of low-level subset API which extends CSS by taking control over styling, layout and render process. It gives direct access to the CSS Object Model (a set of APIs allowing the manipulation of CSS from JavaScript) and allows users to read and modify CSS style dynamically, with JavaScript.
Houdini is only partially supported at present, so is at the very early stage of adoption.

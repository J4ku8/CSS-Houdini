Typed Object Model

The Typed OM is an extension to the existing CSS Object Model (CSSOM) that exposes CSS values as typed JavaScript objects, instead of a simple strings like they are today.

This is code example of Typed Object Model: element.attributeStyleMap.set('opacity', CSS.number(.5));

Element is some variable declared by querrySelector. AttributeStyleMap access CSS of element. Then we can use get or set for work with CSS. In case we would use get, we will get acctual value of property.
